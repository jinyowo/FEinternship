# CSS specificity 계산 순서  

더 특정적인 선택자가 더 일반적인 선택자보다 우선합니다. 가상 엘리먼트와 가장클래스들은 각각 일반적인 엘리먼트와 클래스와 같이 간주됩니다.

* a = 선택자 중 ID의 수
* b = 선택자 중 class의 수
* c = 선택자 중 element의 수
* d = 가상 element 무시

| |a(100자리)|b(10자리)|c(1자리)|선택자 우선순위 계산|
|:---|:--------|:-------|:------|:--------------|
|*{}| 0|0|0|0|
|li{}| 0|0|1|1|
|ul li{}| 0|0|2|2|
|ul ol+li{}| 0|0|3|3|
|li.num{}| 0|1|2|11|
|#wrap{}| 1|0|0|100|
|p#wrap{}| 1|0|1|101|


__선택자 우선순위가 같다면 가장 마지막에 지정된 스타일이 우선 적용.__
